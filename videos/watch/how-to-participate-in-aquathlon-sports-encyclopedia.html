Today, in our sports encyclopedia series we will talk about aquathlon.

Aquathlon is an endurance sport that is a modified and minified version of triathlon consisting of only swimming and running.
The objective is to complete the course in the shortest amount of time.
In Aquathlon, participants compete against each other by completing a course through swimming and running.
Aquathlon has three different variants.
Olympic distance in which participants swim for one mile and then run for further 6.2 miles.
Half Ironman distance in which participants swim for 1.2 miles and then run for further 13.1 miles.
Ironman distance in which participants swim for 2.4 miles and then bike for further 26.2 miles.
There are certain variants in which participants swim for a set duration and then run for a set duration.
The participant that completes the course first or within the given time wins the match.
